# Duranta Online Limited

**Duranta Online Limited** is a modern web application built with **Next.js (App Router)**, **React**, and **Tailwind CSS**. It provides high-speed internet and cable services with a responsive UI, a support center, and an admin dashboard for managing plans, teams, and user notifications.

## Features

- Responsive landing page with animated gradient banner  
- Support center with ticket system  
- Admin dashboard: Plan management, Team management, User notifications  
- Interactive social icons with animations  
- Smooth scroll and WhatsApp contact button  
- Fully client-rendered dynamic components for SSR safety  

---

## Technologies

- **Next.js 16 (App Router)**  
- **React 19**  
- **Tailwind CSS 4**  
- **Framer Motion**  
- **React Icons**  
- **React Slick / Swiper**
